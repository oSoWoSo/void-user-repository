x86_64-musl packages

Add this repository to XBPS

  1. Create the config entry: 
    echo repository=https://repo.osowoso.org/x86_64-musl | sudo tee /etc/xbps.d/00-VUR.conf
  2. Sync and accept fingerprint: 
    sudo xbps-install -S
  3. Install packages: 
    sudo xbps-install <package>
